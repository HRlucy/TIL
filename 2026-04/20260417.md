## 상태값
> [상태 값, 상태 변경 함수] = useState(초기값)
const [textColor, setTextColor] = useState('black');

예시 
function Counter() {
  const [number, setNumber] = useState(0);

  const increaseByTwo = () => {
    setNumber((number) => number + 1);
    setNumber((number) => number + 1);
  };
초기값을 설정한 뒤 함수형식으로 작성

## filter

const updateNames = names.filter(name => name.id !== idToRemove);

return (

## useEffect
> 함수형 컴포넌트에서 렌더링 된 이후 시점에 수행할 내용이 필요한 경우 사용할 수 있는 기능을 hooks으로 제공해주고 있는 useEffect useEffect는 첫 마운트 직후에는 무조건 한 번 실행 됨

렌더링 완료된 후 실행
1. 의존성 배열이 생략된 상태 : 매 렌더링마다 실행
2. 빈 의존성 배열[] : Mount(첫 렌더링) 시에만 실행(주로 초기 데이터 로딩(API호출) 등에 사용)
3. 의존성 배열에 특정 값 지정 [count] : 해당 값이 변경될 때만 실행

Cleanup 함수 : useEffect가 반환하는 함수
컴포넌트가 화면에서 사라지기 직전(unmount)에 호출
