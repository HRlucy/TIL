## DOM(Document Object Model, 문서 객체 모델)
> 브라우저가 HTML 문서를 분석하여 생성하는 계층적 트리 구조의 객체 모델

## `querySelctor`와 `querySelectorAll`
> CSS 선택자를 그대로 사용하여 요소를 찾는, 가장 강력하고 일관성 있는 방법
이 두 가지만 마스터하면 사실상 모든 요소를 선택할 수 있음
과거에는 `getElementById`, `getElementsByClassName` 등을 사용했음
이 메서드들은 `HTMLCollection`이라는 특별한 목록을 반환하는데,
이 목록은 ※'살아있어서' 예기치 못한 문제를 일으킬 수 있음

1. querySelector: CSS 선택자에 맞는 '첫 번째 요소 하나'만 찾아옴
- const $apple = document.querySelector('#apple');

2. querySelectorAll: CSS 선택자에 맞는 '모든 요소'를 NodeList로 찾아옴
- const $fruit = document.querySelectorAll('.fruit');

## 'HTML 가계도' 탐색
> querySelector로 특정 요소를 찾은 뒤, 그 요소를 기준으로 가족(부모, 자식, 형제) 요소를 찾아 나갈 수 있음
눈에 보이지 않는 '공백'이나 '줄바꿈' (텍스트 노드) 때문에 겪는 혼란을 피하기 위해 오직 '요소(Element)'만 찾아주는 프로퍼티들을 사용하는 것이 가장 좋음

1. 눈에 보이지 않는 텍스트 노드(줄바꿈), 주석까지 모두 포함하여 혼란을 줌
- const $family = $family.childNodes;
2. children 모든 '자식 요소'들을 HTMLCollection으로 반환(텍스트 노드 제외)
- const $children = $family.children;
3. firstElementChild 첫 번째 '자식 요소' 반환
- const $first = $family.firstElementChild;
4. lastElementChild 마지막 '자식 요소' 반환
- const $last = $family.lastElementChild;
5. netElementSibling 나의 바로 다음 '형제 요소' 반환
- const $next = $me.nextElementSibling;
6. previousElementSibling 나의 바로 이전 '형제 요소' 반환
- const $prev = $me.previousElementSibling;

## 요소 정보 확인 및 텍스트 조작
> 우리가 선택한 DOM 요소가 어떤 종류인지 확인하고, '요소 안의 텍스트'를 읽고 쓰는 방법 확인

1. 요소의 종류 확인하기
- 모든 DOM 노드는 자신의 종류(요소 노드, 텍스트 노드 등)와 이름을 가지고 있음
2. ※ 요소의 텍스트 조작하기
- 요소 안의 텍스트를 바꾸는 데는 여러 방법이 있지만, textContent가 가장 대중적임
- 요소 노드에 바로 사용 가능
- 요소 안의 모든 자손 노드들의 텍스트까지 전부 가져옴
- 값을 할당하면, 내부의 모든 자식 노드를 지우고 새로운 텍스트로 깔끔하게 교체

## DOM 요소 조작 및 추가하기
1. 부 전체를 갈아엎기 (innerHTML)
> `innerHTML`은 요소 내부의 모든 것을 지우고, 새로운 HTML 문자열로 완전히 교체하는 가장 간단하고 직관적인 방법
-  $area1.innerHTML = '이 글씨로 대체합니다. <strong> 태그는 무시 </strong>';
2. 특정 위치에 살짝 끼워 넣기 (insertAdjacentHTML)
> `insertAdjacentHTML`은 기존 내용은 그대로 유지하면서, 원하는 위치 4곳 중 하나를 지정해 새로운 HTML 요소를 '끼워 넣는' 효율적인 방법
- $area2.insertAdjacentHTML('beforebegin', '<p>div 시작 전(beforebegin)</p>');
3. 정밀하게 요소 만들고 조립하기 (createElement / appendChild)
> 가장 안전하고 정석적인 방법 자바스크립트로 직접 요소(레고 블록)를 만들고, 만들어진 요소를 원하는 위치에 '조립(append)'하는 방식
- const $newLi = document.createElement('li');
- $drinkList.appendChild($newLi);

## 어트리뷰트(Attribute)와 프로퍼티(Property)
> HTML 태그에 부여된 속성(attribute)들을 자바스크립트로 읽고, 쓰고, 확인하고, 지우는 방법
1. const inputValue = $input.getAttribute('value');
2. $input.setAttribute('value','zlafmdud')
3. console.log($input.hasAttribute('class'));

> HTML 어트리뷰트:HTML 문서에 적힌 '초기 설정값'. 마치 건물의 '최초 설계도'
> DOM 프로퍼티:자바스크립트 객체가 가진 '실시간 최신값'. 마치 건물의 '실시간 현황판'
