## 산술 연산자
1. 산술 연산자 피연산자들의 연산 방향은 왼쪽에서 오른쪽
2. 산술 복합 대입 연산자(+=, -=, *=, /=, %=) 왼쪽의 피연산자에 오른쪽 피연산자를 산술한 결과를 왼쪽의 피연산자에 대입
3. 증강 연산자(++, --) '++' : 1 증가 '--' : 1 감소
4. 비교연산자(==, !=, >, <, >=, <=) 피연산자 사이에 상대적 크기를 판단하여 참과 거짓 반환

## 논리 연산자
1. 논리 연결 연산자
- &&(AND) : 두 개의 논리식 모두 참이면 참, 한 개라도 거짓이면 거짓 반환
- || (OR) : 두 개읜 논리식 중 하나라도 참이면 참, 둘 다 거짓인 경우 거짓 반환
2. 논리 부정 연산자
- ! (NOT) : 논리식의 결과가 참이면 거짓을, 거짓이면 참을 반환
3. 논리식 && 논리식 : 앞의 결과가 false이면 뒤를 실행 안함 논리식 || 논리식 : 앞의 결과가 true이면 뒤를 실행 안함
4. 삼항연산자
- (조건식)? true일 때 사용할 값 : false일 때 사용할 값

## 메소드
> 어떤 특정 작업을 수행하기 위한 명령문의 집합
- 메소드 호출 방법 클래스명 사용할 이름 = new 클래스명(); 사용할 이름.메소드명();

## 전달인자(argument)와 매개변수(parameter)
- 메소드를 호출할 때 넘겨주는 값을 '전달인자'라고 하며, 메소드에서 이 값을 받기 위해 선언된 변수를 '매개변수' 라고 함
- 매개변수의 타입, 개수, 순서를 정확히 맞춰서 전달해야함
- void가아닌 다른 타입이 명시된 메소드는 반드시 해당 타입의 값을 반환 해야함 반환된 값은 변수에 저장하거나, 다른 메소드의 전달인자로 즉시 사용 가능
