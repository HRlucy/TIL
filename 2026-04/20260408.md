## Javascript
> 웹 페이지에서 복잡한 기능을 구현할 수 있는 스크립팅 또는 프로그래밍 언어
HTML은 문단, 제목, 데이터 표를 정의하거나 페이지에 이미지와 동영상을 삽입하는 등 웹 콘텐츠를 구성하고 의미를 부여하는 데 사용하는 마크업 언어
CSS는 배경색과 글꼴을 설정하고 콘텐츠를 여러 열에 배치하는 등 HTML 콘텐츠에 스타일을 적용하는 데 사용하는 스타일 규칙 언어
JavaScript는 동적으로 변경되는 콘텐츠를 만들고, 멀티미디어를 제어하고, 이미지에 애니메이션을 적용하는 등 거의 모든 작업을 수행할 수 있는 스크립팅 언어

## 변수에 관하여
> 변수는 데이터를 저장할 때 쓰이는 '이름이 붙은 저장소'이다.
- var : ES5까지 사용했던 유일한 키워드
- let : 값을 변경할 수 있는 변수 선언
- const : 재할당이 금지된 변수 선언

## 변수 규칙
> 변수 이름은 알파벳, 숫자, _, $만 사용할 수 있다.
변수 이름은 숫자로 시작할 수 없다.
카멜케이스(camelCase)로 사용하는 것이 일반적이다.
예약어 사용 금지
대소문자 구분: 변수 이름에서 대소문자를 구분한다. A, a는 다른 변수이다.
의미 있는 이름 사용: 변수 이름은 그 변수의 역할이나 내용을 명확하게 나타내는 것이 좋다.

## 원시 데이터 타입(primitive types)
> 원시 타입은 변경 불가능한 값들이며, 자주 사용되는 원시 타입으로는 숫자(number), 문자열(string), 불린(boolean), null, undefined 등이 있다

- 1. 숫자(number)
자바의 경우 정수와 실수를 구분해 int, long, float, double 등과 같은 다양한 숫자 타입을 제공
== 값이 같을 때 비교
=== 값과 타입이 모두 같을 때 비교

- 2.  문자열(String)
문자열 타입은 텍스트 데이터를 나타내는데 사용
작은 따옴표(''), 큰 따옴표(""), 또는 백틱(``)으로 텍스트를 감싼다.

- 3. 불린(boolean)
논리적 참, 거짓을 나타내는 true 와 false 뿐이다.

- 4. null
null 타입의 값은 null이 유일
명시적으로 값이 없음을 의도적으로 명시할 때 사용

- 5. undefined
undefined는 자바스크립트 엔진이 변수를 초기화 할 때 사용하는 값이므로 개발자
가 의도적으로 변수에 할당하는 것은 본래 취지와 어긋나고 혼란을 줄 수 있으므로
지양
변수에 값이 할당되지 않았을 때 자동적으로 할당되는 값

## 정적 타입(static/strong type) 언어(c++, java)
> 변수의 타입을 변경할 수 없으며, 변수에 선언한 타입에 맞는 값만 할당할 수 있음

## 동적 타입(dynamic/weak type) 언어(javascript, python)
> 자바스크립트는 var, let, const 키워드를 사용해 변수를 선언할 뿐 데이터 타입을 사전에 선언하지 않음


## 암묵적 타입 변환
> 자바스크립트 엔진에 의해 암묵적으로 타입이 자동 변환 되는 것 타입 변환 된 결과를 예측할 수 있어야 오류를 방지할 수 있음
- 예시)
문자열 타입 변환
console.log(10 + '20');
console.log(1 + '');
console.log(NaN + '');
console.log(undefined + '');

숫자 타입 변환
console.log(10 - '5');
console.log(10 * 5);
console.log(10 / 5 );

## 명시적 타입 변환
> 개발자의 의도에 따라 값의 타입을 변환하는 것
- 예시)
문자열 타입 변환
console.log(String(10));
console.log(String(true));
console.log(null);
console.log((10).toString());

숫자 타입 변환
console.log(Number('10.01'));
console.log(Number('10원'));
console.log(Number(true));
console.log(Number(false));
console.log(parseInt('10.01'));
console.log(parseFloat('10.01'));

논리 타입 변환
console.log(Boolean('JS'));
console.log(Boolean(0));
console.log(Boolean(NaN));
console.log(!true);
console.log(!!'');
console.log(!!1);
console.log(!!0);

## 비교 연산자
> 비교 연산자는 좌항과 우항의 비교 연산자를 비교한 다음 그 결과를 불리언 값으로 반환 if문이나 for문 같은 제어문의 조건식에서 주로 사용한다
- 1. 동등 연산자(==, !=) 먼저 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교
console.log(x == y);
console.log(x != y);
console.log(x == '10');

- 2. 일치 연산자(===, !==) 타입과 값이 모두 일치하는지 비교
console.log(x === 10);
console.log(x === '10');

- 3. 대소 비교 연산자
console.log(x > y);
console.log(x <= y);

## 논리 연산자
> 논리 연산자는 여러 조건을 결합하여 하나의 논리적 결과를 도출한다 AND, OR, NOT 연산자가 있다
- 1. 논리 AND (&&) 모두 true 일 때만 true 변환
- 2. 논리 OR (||) 둘 중 하나만 true여도 true 반환
- 3. 논리 NOT (!) true면 false로, false면 true로 반전

## if 조건문
if (조건식) {
  조건식이 true인 경우 실행구문
} else (조건식) {
  조건식이 false인 경우 실행구문
}

## 조건문 switch문
switch(조건식) {
  case(변수a) :
  변수a일 경우 실행구문
  case(변수b) :
  변수b일 경우 실행구문
  case(변수c) :
  변수c일 경우 실행구문
}

## for 반복문
명확한 반복 횟수를 지정하여 코드 실행
for (초기화; 조건식; 증감식) {
  반복 실행할 코드
}
