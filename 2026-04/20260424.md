## zustand
> 독일어로 '상태'라는 뜻으로 React 생태계에서 사용하는 상태 관리 라이브러리

## zustand가 필요한 이유
> 데이터를 전달하는 과정에서 거쳐야 하는 컴포넌트가 너무 많은 상황이 있어, 상태는 여러 페이지, 프로덕트 전반에 걸쳐 공통으로 사용하는 경우가 있기 때문에 상태 관리 라이브러리가 등장하게 되었음

npm install zustand로 설치

import { create } from 'zustand'

export const useStore = create((set) => ({
  bears: 0,
  increasePopulation: () => set((state) => ({ bears: state.bears + 1 })),
  removeAllBears: () => set({ bears: 0 }),
  updateBears: (newBears) => set({ bears: newBears }),
}))
